Super Othello

A simple othello player made for an University project. The original version provided also an online mode with a simple matchmaking system based on mySQL and PHP.

Zx Spectrum 48k

An emulator for the famous Speccy! It is possible to load .tap and .z80 programs.

Pacman Arcade

An emulator for pacman cabinet and some of its clone! Main board consists of:

  • Z80 Cpu
  • Namco Wave Sound Generator
  • Tile based background
  • Hardware sprites

Space Invaders

An emulator of the Midway 8080 Cabinet. At the moment only Space Invaders is supported. Main board consists of:

  • Intel 8080 Cpu
  • Bitmap black and white display
  • A couple of fixed sound

エミュボーイ

A simple GameBoy emulator. This project is not finished but it can emulate almost all games.

  • Accurate interrupt emulation
  • Cycle accurate cpu core
  • Basic support of link cable via LAN
  • support for .sav files
  • A large variety of MBCs supported
    • MBC1
    • MBC2
    • MBC3
    • MBC5
    • MMM01
    • M161
    • MBC1M
